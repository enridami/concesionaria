## Sistema Concesionaria de vehículos
Sistema CRUD para un sistema de registro de vehiculos de una concesionaria. En el se aplico los conceptos de Create, Read, Update y Delete dentro de una base de datos en MySQL.

## Objetivos
El proyecto busca reforzar los conocimientos sobre POO(Programación Orientada a Objetos) dentro de un entorno de desarrollo integrado (NetBeans), además de entender cómo funciona en JAVA el sistema JPA, uniendo la IGU, la lógica, la persistencia y la base de datos. Se aplico una separación de capas para cada tarea correspondiente, con archivos delimitados a su función (Controladora, ControladoraPersistencia, etc).

## Tecnologias utilizadas
- Java
- Java JPA ORM (Java Persistence Api)
- Xampp
- MySQL
- Eclipse
